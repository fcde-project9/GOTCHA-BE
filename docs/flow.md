# 서비스 플로우

## 화면 플로우

```
스플래시 (0-001)
    ↓
소셜 로그인 (0-002)
    ↓
닉네임 생성 (0-003)
    ↓
메인 지도 (1-001)
    ├── 지역 검색 (2-001, 2-002)
    ├── 가게 상세 (1-002~007)
    ├── 제보하기 (4-001~005)
    ├── 찜 목록 (3-001)
    └── 마이페이지 (6-001~002)
```

---

## 검색 플로우

```
1. 사용자가 "신사" 입력
2. 프론트: 카카오 지도 API로 지역 자동완성
3. "신사역" 선택 → 좌표 획득
4. 백엔드: GET /api/shops?lat=&lng=&radius=
5. 주변 가챠샵 목록 반환 (거리순 정렬)
6. 지도 이동 + 마커 표시 + 리스트 표시
```

**핵심**: 지역 검색은 **프론트(카카오 API)**, 가챠샵 조회는 **백엔드**

---

## 메인 화면 구성

- 상단: 검색바
- 중앙: 지도 + 마커 ("나" 분홍, "가챠샵" 검정)
- 하단: 가게 리스트 (영업중 뱃지, 거리, 찜 버튼)
- 탭바: 홈, 제보, 찜, 마이
